Denna app beräknar medelvärden för betyg.

UTF-8 gäller.

Skapa filerna 
* index.html
* sketch.js

## Input

En sträng med bokstäver.

A = 20  
E = 10  
F = 0  

Små bokstäver innebär termin istället för läsår.

Skriv en readme.md

Detta fält ska ha fokus.

## Output

Exempel:  

AB => (20 + 17.5) / 2 = 37.5/2  
aBb => (20/2 + 17.5 + 17.5/2) / (0.5 + 1 + 0.5)  
CAAAAabCAACDCBAABa => 17.95
