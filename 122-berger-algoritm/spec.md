Skapa berger.js.

berger.js ska producera lottning för ett visst, jämnt, antal spelare.

# Indata:

n : antal spelare
rond : 1 .. n-1

# Utdata:

Bordlista som visar vilka som ska mötas

# Exempel 1:

n = 8
rond = 1

3-6
2-7
4-5
1-8

# Exempel 2:

n = 8
rond = 2

3-7
4-6
2-1
5-8

n = 10
rond = 1

4-7

5-6
1-10
