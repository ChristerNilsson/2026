# Porträtt Bokmärke

Bokmärket visar turneringstabeller från `member.schack.se` i ett kolumnindelat läge.  
Detta är användbart vid stora turneringar och när skärmar inte går att vrida till porträttläge.  

## Installera

1. Skapa ett nytt bokmärke i webbläsaren.
2. Ge bokmärket ett namn, till exempel `Porträtt`.
3. Kopiera hela innehållet i `bookmarklet.txt`.
4. Klistra in koden som bokmärkets URL/adress.
5. Spara bokmärket.

## Använda

1. Öppna turneringssidan, till exempel:
   `https://member.schack.se/ShowTournamentServlet?id=15056`
2. Välj rond som vanligt.
3. Klicka på bokmärket.
4. Sidan ersätts av en enda tabell.

## Tangenter

- `↑` och `↓`: byter tabell.
- `←` och `→`: minskar eller ökar antal kolumner.
- `x`: stänger bokmärkets vy.

## Visning

Bokmärket försöker behålla tabellernas ursprungliga utseende, men gör vissa justeringar för att spara plats.
