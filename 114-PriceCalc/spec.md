Produce sketch.js and index.html

Input:

* A = Belopp
* B = Steg

Output:

* O1 = Första pris
* O2 = Andra pris
* O3 = Tredje pris

Calculations

* O1 = A / 3 + B
* O2 = A / 3
* O3 = A / 3 - B

All outputs are rounded up to nearest 100

Examples:

* A = 2000
* B = 100

* O1 = 800
* O2 = 700
* O3 = 600
